# Bibliotecas Necessárias

Esta pasta contém todas as bibliotecas necessárias para o correto funcionamento do código da **Chocadeira Automática**.

Incluem:

- Bibliotecas de sensores (temperatura, humidade, etc.)
- Bibliotecas de comunicação (I2C, Serial, etc.)
- Dependências externas utilizadas no código final

Estas bibliotecas devem ser instaladas no Arduino IDE antes de abrir e carregar o ficheiro principal.

Todos os ficheiros aqui presentes garantem que o código funcione sem erros de dependências e estão preparados para o sistema físico da chocadeira.
