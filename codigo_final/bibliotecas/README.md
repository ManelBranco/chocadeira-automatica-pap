#Bibliotecas Necessárias

Esta pasta contém todas as bibliotecas necessárias para que o código da Chocadeira Automática funcione corretamente no Arduino.

Inclui:

Bibliotecas de sensores (temperatura, humidade, etc.)

Bibliotecas de comunicação (I2C, serial, etc.)

Qualquer dependência externa utilizada no código final

Estas bibliotecas devem ser instaladas no Arduino IDE antes de abrir e carregar o ficheiro principal do projeto.

Tudo está organizado para facilitar a instalação e garantir que o código funcione sem erros de dependências.
