# Código de Teste e Desenvolvimento

Esta pasta contém diferentes versões experimentais do código desenvolvidas ao longo do projeto.

Nem todos os ficheiros aqui presentes se encontram totalmente funcionais, pois representam fases de teste, validação de componentes e ajustes de lógica.

Incluem:

- Testes individuais de sensores
- Ensaios de controlo de relés
- Experiências com motores e temporizações
- Versões intermédias da lógica de controlo

Estes ficheiros demonstram o processo evolutivo do projeto e o trabalho realizado até atingir a versão final estável.
